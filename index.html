CSS: DISPLAY & POSITION

Position

Property value Description
Position Static Default value. Elements render in order, as they appear in the document flow.
Position relative Element is positioned relative to its normal position. You can use rulest to change its position.
Position Absolute It is removed from the normal flow of the document, therefore it is ignored by other elements also it overlaps them. its positioned relative to its nearest positioned ancestor otherwise its positioned to the body of the document.
Position Fixed It is removed from the normal flow of the document. It is positioned to the initial container block of the element is the viewport, it dosen't affects the document flow its position can be set by ruleset. This values always create a new stacking context.
Position Sticky Hybrid of fixed and relative. It is treated as a element with relative position until it crosses a specified threshold eg top: 10px, at wich point is treated as fixed until it reaches the limit of his parents

Display

Property Value Description
Display Inline it does not starts a new line. It takes only the necessary space for its content. It can not be changed its position by ruleset
Display Block Always starts a new line and It can takes all the available width in the line.
Display Inline-block It dosen't start a new line and other elements cab be set next to the element. It can takes all the space available with ruleset
